Classification of Urdu News Articles

This project applies supervised machine learning to classify Urdu-language news articles into five categories: entertainment, business, sports, science-technology, and international. From 2,750 articles scraped from Urdu news websites, preprocessing included normalization, lemmatization, and tokenization. Three models were evaluated: Multinomial Naive Bayes (MNB), Logistic Regression, and Neural Networks. MNB achieved 96.55% accuracy, Logistic Regression 95.27%, and the Neural Network outperformed both with 97.45%, using advanced modeling and dropout layers to prevent overfitting. Performance was assessed through accuracy, precision, recall, and F1 scores, with confusion matrices highlighting misclassifications. The project showcases machine learning’s potential in underrepresented languages like Urdu while noting limitations such as reliance on traditional models and limited semantic understanding.

Neural Networks for Detecting Code Rot in Humanoids

This project develops a neural network to identify malfunctioning humanoid Guardians affected by "Code Rot". Using a dataset of 8,580 CT scan images labeled as infected or not, the binary classification model predicts Code Rot to ensure the safety of Earth Junior's inhabitants. The model architecture includes fully connected layers with dropout and ReLU activations, optimized using binary cross-entropy loss and Adam optimizer. Data preprocessing involves image resizing, grayscale conversion, and normalization, with a 60:20:20 train-validation-test split. Training progress is tracked with accuracy and loss metrics. Achieving high accuracy, the model is evaluated through confusion matrices, classification reports, and ROC curves. Misclassified cases are analyzed to refine performance. The project demonstrates the effectiveness of neural networks in solving futuristic challenges.

Implementing and Evaluating KNN and RNN Classifiers

This project focuses on implementing and evaluating K-Nearest Neighbors (KNN) and Radius Nearest Neighbors (RNN) classifiers using both manual implementation and `scikit-learn`. The first part applies KNN to the MNIST dataset of handwritten digits, with data preprocessing and normalization. Two distance metrics—Euclidean and Manhattan—are used, and k-fold cross-validation determines the optimal k . Evaluation metrics include confusion matrices, accuracy, and macro-average F1 scores. The second part explores RNN on the Breast Cancer dataset to classify tumors as malignant or benign. RNN uses a radius-based approach to identify neighbors, with cross-validation conducted for varying radius values to optimize performance. Predictions on the test data highlight RNN's limitations for high-dimensional datasets like MNIST, emphasizing challenges such as the curse of dimensionality.

Human-Zombie Screening: Linear and Logistic Regression

This project centers on developing machine learning models for screening and classifying individuals based on their human-zombie score, using a dataset with features like height, weight, and activity levels. The linear regression component predicts the human-zombie score as a continuous variable, implementing multivariate regression from scratch and using scikit-learn for comparisons. Techniques like gradient descent and regularization methods (Ridge, Lasso, and ElasticNet) are employed, with model performance evaluated through mean squared error. The logistic regression part classifies individuals into categories: "Human," "Needs Further Testing," or "Zombie," based on thresholded scores. Implementations include a manual regularized logistic regression model and scikit-learn’s version. The project incorporates multiclass classification using the One-vs-All approach, with metrics such as accuracy, F1 score etc reported.

Implementing and Evaluating Naive Bayes Classifiers

This project explores the implementation and evaluation of Naive Bayes classifiers for two datasets: a Golf dataset and the Tweet Evaluation dataset. The Golf dataset predicts whether golf will be played based on binary and categorical features, utilizing the Bernoulli Naive Bayes classifier. The Tweet Evaluation dataset performs sentiment analysis on tweets, classifying them into emotional categories using the Multinomial Naive Bayes classifier. The project includes both manual and `scikit-learn` implementations of Naive Bayes models, with detailed preprocessing steps such as one-hot encoding for the Golf dataset and text cleaning for the Tweet Evaluation dataset. Metrics like accuracy, precision, recall, and F1 score are reported for both, comparing their performance. The study emphasizes the strengths of each Naive Bayes variant, highlighting their applicability to different problem domains.
